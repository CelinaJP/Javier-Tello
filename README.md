# Javier Tello | Portfolio de Edición de Video

Portfolio profesional de Javier Tello, editor de video especializado en crear contenido audiovisual de alto impacto para marcas, creadores y emprendedores. Este proyecto presenta su experiencia, servicios, testimonios y un canal directo de contacto para trabajar juntos.

## Sobre mí

Javier Tello es un editor de video con más de 10 años de experiencia en la creación de contenidos visuales que conectan con la audiencia. Su enfoque combina narrativa, ritmo, detalle y estética para transformar ideas en piezas profesionales y memorables.

Su trabajo está orientado a proyectos que buscan:

- mejorar la identidad visual de una marca o creador
- aumentar la retención de la audiencia
- convertir contenido en piezas más dinámicas y persuasivas
- entregar videos con una estética cinematográfica y moderna

## Servicios

El portfolio destaca sus principales áreas de especialización:

- Contenido corto para Instagram Reels y TikTok
- Contenido largo para YouTube, podcasts y entrevistas
- Presentaciones y videos promocionales para speakers y marcas personales
- Edición de VSL (Video Sales Letter) con enfoque comercial

## Características del proyecto

- Diseño elegante y premium con estética oscura y sofisticada
- Secciones claras para presentación personal, servicios y testimonios
- Diseño responsive para dispositivos móviles y escritorio
- Integración visual con elementos de alto impacto y animaciones suaves
- Enlaces directos a Instagram y WhatsApp para contacto rápido

## Tecnologías utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome
- AOS (animaciones al hacer scroll)

## Cómo ver el proyecto

1. Abre el archivo [javier-tello-landing.html](javier-tello-landing.html) en tu navegador.
2. Si deseas una experiencia más cercana a un sitio web real, puedes servir la carpeta localmente con un servidor simple como:

```bash
python3 -m http.server 8000
```

Luego ingresa a http://localhost:8000 en tu navegador.
